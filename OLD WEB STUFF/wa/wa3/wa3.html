Unity

What is Unity?

Unity is a cross platfom game engine that is developed by Unity Technologies.

This engine has support on a variety of platforms such as desktop, mobile, console, and virtual reality.

Unity is a great place for beginner developers and extreamly popular for creating indie games.

The game engine itself can be used to create 3D and 2D games, as well as be used to create interactive simulations and other experiences.

Features that come with Unity

Unity gives the users the ability to create games and experiences in both the 2D and the 3D realms.

With a great documentation for refrence unity allows the users to engage in scripting with C#.

For anyone who might be new to the game engine and scripting as a whole unity has a huge selection to start learning at your own pace such as Unitys tutorial Create With Code.

Don't worry not everything with Unity is just code, Unity has a user friendly built in editor that allows the user to plug and play at ease.

For a much better experience with Unity, they also allow developers to create thier own textures and aniations from scracth allowing any developer to get as creative as they want.

If you ever feel unsure as to what animation or texture to use Unity also allows for the users to use anything provided in the built in asset store, each asset created by Unity or even other developers

Supported Platforms

Unity is a cross-platform engine. The editor is supported on Windows, MacOS, and Linux.

Unity currently supports building games on the following platforms

  • Mobile: IOS, Andriod

  • Desktop: Windows, Mac, Linux

  • Web: WebGL

  • Console: PlayStation 4, PlayStation 5, Xbox One/S, Xbox Series X/S, Nintendo Switch

  • Vitrual Reality: Oculus, PlayStation VR, Windows Mixed Reality, Steam VR
